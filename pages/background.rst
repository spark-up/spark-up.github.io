Background
==========
Over the past decade there has been extensive research done on automated Machine Learning or
AutoML. New AutoML platforms like Google's Cloud AutoML and Saleforce's Einstein has 
automated the entire ML pipeline. However, there has been lackluster research done on 
automated data preparation.
